# Steps

Use this checklist before and during slide creation.

## Step 0: Get Perspective
- [ ] Remember the audience knows less context than you.
- [ ] Reframe the topic from the audience point of view.

## Step 1: Whiteboard First
- [ ] Explain the work at a whiteboard to one colleague.
- [ ] Watch reactions and keep the wording/drawing that makes it click.
- [ ] Repeat with another colleague and improve the flow.

## Step 2: Set Goals
- [ ] Write 3 technical points the audience must understand.
- [ ] Write 3 messages the audience should remember.
- [ ] Make these messages explicit in examples and time use.

## Step 3: Build an Outline
- [ ] Draft section structure for the talk.
- [ ] Check that each section flows naturally to the next.

## Step 4: Storyboard on Paper
- [ ] Sketch one slide per paper page.
- [ ] Talk while sketching and revise quickly.
- [ ] Reorder or replace pages until the sequence works.

## Step 5: Create Digital Slides
- [ ] Convert the storyboard to digital slides.
- [ ] Keep the first version simple and fill details later.
- [ ] Get feedback and apply improvements.

## Step 6: Rehearse Alone
- [ ] Practice with a timer multiple times.
- [ ] Keep short notes for clear phrases that work.
- [ ] Fix slides that are hard to explain.

## Step 7: Practice with Audience
- [ ] Present to colleagues, including newcomers.
- [ ] Collect feedback and revise story/slides.
- [ ] If possible, run a second practice talk.

## Step 8: Final Timing and Polish
- [ ] Finalize wording and delivery.
- [ ] Practice standing (mirror optional).
- [ ] Confirm the talk fits the target time.

## Goal
- [ ] Deliver a clear and engaging presentation.
